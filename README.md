## Clone this project

```bash
> pkg update && pkg upgrade
> pkg install bash
> pkg install git
> git clone https://github.com/RTeam1/wabot2
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd wabot2
> bash install.sh
```

### Usage
```bash
> npm start
```

### if npm Error?
```bash
> node index.js
```
